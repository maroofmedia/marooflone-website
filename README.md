## 📖 About The Repo

This repository contains the source code for my personal website, powered by **Eleventy (11ty)**. While the original site was built with standard HTML/CSS/JS, this version leverages the power of a static site generator to deliver a seamless blogging experience while maintaining a minimal footprint.

### Key Features

  * ⚡ **Blazing Fast:** Static HTML generation ensures top-tier performance.
  * 🎨 **Minimalist & Responsive:** A clean design that works perfectly on all devices.
  * ✍️ **Markdown Support:** Blog posts are written in Markdown for easy content management.
  * 🔍 **SEO Optimized:** Built with search engines in mind.
  * 🗂️ **Organized Structure:** scalable file architecture.

## 🛠️ Built With

  * **[Eleventy (11ty)](https://www.11ty.dev/)** – Static Site Generator
  * **HTML5** – Semantic structure
  * **CSS3** – Custom, lightweight styling
  * **JavaScript** – Minimal client-side interactivity
  * **Nunjucks** – Templating engine

## 📂 Project Structure

```text
├── src/
│   ├── _includes/       # Nunjucks templates (Header, Footer, Layouts)
│   ├── admin/           # Netlify CMS configuration
│   ├── assets/          # Images, fonts, and optimized static files
│   ├── blog/            # Blog posts (Markdown files)
│   ├── style.css        # Refactored main stylesheet
│   └── index.njk        # Homepage layout
├── .eleventy.js         # Eleventy config
├── package.json         # Dependencies and scripts
└── README.md            # Documentation
```

## 💻 Getting Started

To run this project locally and see the v2 updates in action:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/maroofmedia/marooflone-website.git
    ```
2.  **Navigate into the project folder:**
    ```bash
    cd marooflone-website
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Start the development server:**
    ```bash
    npm start
    ```

Your site should now be running at `http://localhost:8080`.
